---
title: "Deprecate a template"
openapi: "POST /v1/templates/{template_id}/deprecate"
---

Mark a template as deprecated. Deprecated templates remain accessible but should not be used for new jobs. Existing jobs using the template are not affected.

<Panel>
  <Warning>
    **Deprecation is Permanent**

    Once deprecated, a template cannot be un-deprecated. Create a new version instead.
  </Warning>

  <Tip>
    Use versioning (`version` field) to create updated templates while keeping old ones deprecated.
  </Tip>
</Panel>

<RequestExample>

```bash cURL
curl -X POST "https://api.peepal.dev/v1/templates/tpl_01ARZ3NDEKTSV4RRFFQ69G5FAV/deprecate" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

template_id = "tpl_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/templates/{template_id}/deprecate"
headers = {"x-api-key": "YOUR_API_KEY"}

response = requests.post(url, headers=headers)
template = response.json()
```
