---
title: "Cancel a job"
openapi: "POST /v1/jobs/{job_id}/cancel"
---

Cancel a job that is currently queued or running. Completed or failed jobs cannot be canceled.

## When to Cancel

- Job was created by mistake
- Requirements changed
- Cost control (stop expensive verification)
- User requested cancellation

<Panel>
  <Warning>
    **Cancellation is Immediate**
    
    Once canceled, the job cannot be resumed. Any partial results are discarded.
  </Warning>
  
  <Tip>
    You can only cancel jobs in "queued" or "running" status.
  </Tip>
</Panel>

<RequestExample>

```bash cURL
curl -X POST "https://api.peepal.dev/v1/jobs/job_01ARZ3NDEKTSV4RRFFQ69G5FAV/cancel" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

job_id = "job_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/jobs/{job_id}/cancel"
headers = {"x-api-key": "YOUR_API_KEY"}

response = requests.post(url, headers=headers)
canceled_job = response.json()
```
