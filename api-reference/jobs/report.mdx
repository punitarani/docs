---
title: "Get a report"
openapi: "GET /v1/jobs/{job_id}/report"
---

Retrieve the verification report for a completed job. This includes the overall score, claim results, identity resolution, and all evidence sources.

## Report Availability

Reports are only available for jobs with `status: "completed"`. Attempting to fetch a report for an incomplete job will return a 409 Conflict error.

<Panel>
  <Info>
    **Understanding the Report**

    - **score**: Overall verification score (1-5)
    - **score_breakdown**: Scores by category (identity, employment, education, business), each 1-5
    - **identity_resolution**: How well we matched the person
    - **claims**: Individual claim verification results
    - **sources**: All evidence used in verification
    - **confidence values**: All confidence fields use a 1-5 scale
  </Info>

  <Tip>
    Each claim includes an `explanation` field describing why it was verified or not.
  </Tip>
</Panel>

<RequestExample>

```bash cURL
curl -X GET "https://api.peepal.dev/v1/jobs/job_01ARZ3NDEKTSV4RRFFQ69G5FAV/report" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

job_id = "job_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/jobs/{job_id}/report"
headers = {"x-api-key": "YOUR_API_KEY"}

response = requests.get(url, headers=headers)
report = response.json()

print(f"Score: {report['score']}/5")
for claim in report['claims']:
    print(f"{claim['claim_key']}: {claim['status']}")
```


```typescript TypeScript
interface Report {
  job_id: string;
  status: 'completed';
  score: number;
  score_breakdown: {
    identity: number;
    employment: number;
    education: number;
    business: number;
  };
  identity_resolution: {
    status: 'matched' | 'ambiguous' | 'no_match';
    confidence: number;
    candidates: Array<{
      label: string;
      confidence: number;
      profile_urls: string[];
    }>;
    identity_status?: 'confirmed' | 'high_confidence' | 'likely' | 'possible' | 'no_match';
    identity_confidence?: number;
    profiles?: Array<{
      url: string;
      type: 'linkedin' | 'github' | 'personal_website' | 'portfolio' | 'twitter' | 'company_page' | 'other';
      title: string | null;
      is_primary: boolean;
    }>;
  };
  claims: ClaimResult[];
  sources: Source[];
  created_at: string;
}

const jobId = 'job_01ARZ3NDEKTSV4RRFFQ69G5FAV';
const url = `https://api.peepal.dev/v1/jobs/${jobId}/report`;

const response = await fetch(url, {
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const report: Report = await response.json();
```
