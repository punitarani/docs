---
title: "Get a webhook endpoint"
openapi: "GET /v1/webhooks/endpoints/{endpoint_id}"
---

Retrieve the details of a specific webhook endpoint.

<Panel>
  <Info>
    Use this to verify endpoint configuration and check which events are subscribed.
  </Info>
</Panel>

<RequestExample>

```bash cURL
curl -X GET "https://api.peepal.dev/v1/webhooks/endpoints/wh_01ARZ3NDEKTSV4RRFFQ69G5FAV" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

endpoint_id = "wh_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/webhooks/endpoints/{endpoint_id}"
headers = {"x-api-key": "YOUR_API_KEY"}

response = requests.get(url, headers=headers)
endpoint = response.json()
```

```javascript JavaScript
const endpointId = 'wh_01ARZ3NDEKTSV4RRFFQ69G5FAV';
const url = `https://api.peepal.dev/v1/webhooks/endpoints/${endpointId}`;

const response = await fetch(url, {
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const endpoint = await response.json();
```
