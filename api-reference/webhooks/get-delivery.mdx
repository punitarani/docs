---
title: "Get a webhook delivery"
openapi: "GET /v1/webhooks/deliveries/{delivery_id}"
---

Retrieve the details of a specific webhook delivery attempt.

<Panel>
  <Info>
    Use this to investigate webhook delivery issues and verify successful deliveries.
  </Info>
</Panel>

<RequestExample>

```bash cURL
curl -X GET "https://api.peepal.dev/v1/webhooks/deliveries/del_01ARZ3NDEKTSV4RRFFQ69G5FAV" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

delivery_id = "del_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/webhooks/deliveries/{delivery_id}"
headers = {"x-api-key": "YOUR_API_KEY"}

response = requests.get(url, headers=headers)
delivery = response.json()

if delivery["status"] == "failure":
    print(f"Delivery failed: {delivery['error']}")
```
