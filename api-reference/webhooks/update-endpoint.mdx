---
title: "Update a webhook endpoint"
openapi: "PATCH /v1/webhooks/endpoints/{endpoint_id}"
---

Update the configuration of an existing webhook endpoint. You can modify events, enable/disable status, and description.

<Panel>
  <Info>
    **Updatable Fields**

    - `events`: Change which events trigger webhooks
    - `enabled`: Enable or disable the endpoint
    - `description`: Update the description
  </Info>

  <Tip>
    The URL cannot be updated. Create a new endpoint if you need a different URL.
  </Tip>
</Panel>

<RequestExample>

```bash cURL
curl -X PATCH "https://api.peepal.dev/v1/webhooks/endpoints/wh_01ARZ3NDEKTSV4RRFFQ69G5FAV" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "events": ["job.completed", "job.failed", "job.started"],
    "enabled": true
  }'
```

```python Python
import requests

endpoint_id = "wh_01ARZ3NDEKTSV4RRFFQ69G5FAV"
url = f"https://api.peepal.dev/v1/webhooks/endpoints/{endpoint_id}"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    "events": ["job.completed", "job.failed", "job.started"],
    "enabled": True
}

response = requests.patch(url, json=payload, headers=headers)
endpoint = response.json()
```

```javascript JavaScript
const endpointId = 'wh_01ARZ3NDEKTSV4RRFFQ69G5FAV';
const url = `https://api.peepal.dev/v1/webhooks/endpoints/${endpointId}`;

const response = await fetch(url, {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    events: ['job.completed', 'job.failed', 'job.started'],
    enabled: true
  })
});

const endpoint = await response.json();
```
